安装Hexo前需要安装：

  • Node.js（建议10.0以上版本）
  • Git

命令执行完成后，将在根目录下生成相应的文件，我们需要关注下述几个目录及文件：

  • _config.yml：存放配置信息，可以在此配置大部分的参数
  • source->posts：存放post的md文件
  • themes：主题文件夹

可以简写为$ hexo s，用于启动服务器。默认情况下，访问网址为： http://localhost:4000/。
